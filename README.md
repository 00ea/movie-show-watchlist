# **Movie / Show Watchlist**
A watchlist for TV shows and movies that users can use with full CRUD functionality

# **CRUD Functions**

Create: add movies / shows to watchlist (mongodb) <br/>
Read: view watchlist or watch history <br/>
Update: mark movies / shows as watched & add your own ratings & reviews to them <br/>
Delete: Remove movies from watchlist and clear old entries 

# **Extra Features**

- Searching with sorting & filtering for ease of access <br/>
- Will display a thumbnail for the movie (if possible) <br/>
- A card-like format similar to the one Netflix has so things are very easily visible <br/>
- Color coding to make it easier to discern <br/>
    - Red = Not started <br/>
    - Yellow = In progress <br/>
    - Green = Completed <br/>
- Random button to select a random movie to watch <br/>
- Dashboard showing stats for each show / movie
